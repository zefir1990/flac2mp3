# flac2mp3

[**🇬🇧 Read in English**](README.md)

Скрипт на Python для рекурсивного конвертирования аудиофайлов FLAC в формат MP3 высокого качества с использованием `ffmpeg`.

## Требования

- **Python 3.x**
- Установленный **ffmpeg**, добавленный в системную переменную PATH.

## Возможности

- Рекурсивный поиск всех файлов `.flac` в указанной директории.
- Кодирование MP3 с переменным битрейтом (VBR) высокого качества с использованием кодека `libmp3lame` (уровень качества `-q:a 0`).
- Корректная обработка пробелов в именах файлов и папок.
- Безопасный запуск и обработка процессов.
- Несколько режимов экспорта:
  - **In-place (На месте)**: сохраняет `.mp3` рядом с исходным `.flac`.
  - **Сохранение структуры директорий**: воссоздает дерево папок источника в целевой папке.
  - **Плоский экспорт (Без структуры)**: экспортирует все MP3 напрямую в одну указанную папку.
  - **Комбинированный**: позволяет использовать оба режима экспорта одновременно. Файл конвертируется один раз, после чего копируется, что позволяет избежать повторного кодирования.
- **Параллельный запуск**: многопоточная конвертация для ускорения процесса.

## Использование

Запуск скрипта из командной строки:

```powershell
python flac2mp3.py [директория_поиска] [параметры]
```

Если `[директория_поиска]` не указана, по умолчанию используется текущая рабочая директория (`.`).

# Реальный пример использования

Конвертация всех flac-файлов из `D:\Downloads` в `D:\mp3s` с использованием 4 параллельных потоков и сохранением структуры папок:

```powershell
python .\flac2mp3.py D:\Downloads\ --output-directory-save-directories-tree=D:\mp3s\ --parallel-run=4
```

### Параметры

- `--output-directory-save-directories-tree=<путь>` (или без `--`: `output-directory-save-directories-tree=<путь>`)
  
  Экспортирует файлы MP3 в указанный `<путь>` с воссозданием структуры подкаталогов.

- `--output-directory-without-directories-tree=<путь>` (или без `--`: `output-directory-without-directories-tree=<путь>`)
  
  Экспортирует все файлы MP3 напрямую в `<путь>` без сохранения структуры подкаталогов.

- `--parallel-run=<число>` (или без `--`: `parallel-run=<число>`)
  
  Количество одновременно запущенных конвертаций (по умолчанию: `1`).

### Примеры

#### 1. Конвертация на месте
Конвертировать все файлы FLAC в текущей папке и сохранить MP3 файлы рядом с исходными:
```powershell
python flac2mp3.py
```

#### 2. Воссоздание структуры папок в другом месте
Конвертировать файлы FLAC из `C:\Music\FLAC` и экспортировать их в `D:\Music\MP3` с сохранением дерева папок:
```powershell
python flac2mp3.py C:\Music\FLAC output-directory-save-directories-tree=D:\Music\MP3
```

#### 3. Плоский экспорт
Конвертировать файлы FLAC из `C:\Music\FLAC` и поместить все сгенерированные файлы MP3 в папку `D:\Music\FlatMP3`:
```powershell
python flac2mp3.py C:\Music\FLAC output-directory-without-directories-tree=D:\Music\FlatMP3
```

#### 4. Комбинированный режим экспорта
Конвертировать файлы FLAC и сохранить их одновременно и в структурированную, и в плоскую директорию:
```powershell
python flac2mp3.py C:\Music\FLAC output-directory-save-directories-tree=D:\Music\MP3 output-directory-without-directories-tree=D:\Music\FlatMP3
```

#### 5. Параллельная конвертация
Конвертировать файлы FLAC в параллельном режиме с использованием 4 потоков:
```powershell
python flac2mp3.py parallel-run=4
```
